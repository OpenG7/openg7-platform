# 🖥️ OpenG7 Platform

**Technical repository for the OpenG7 project**  
This repository contains the source code, technical documentation, and roadmap for the development of the OpenG7 platform.

---

## 📦 Stack technologique / Tech Stack

- **Frontend**: Angular 19.2.5 (Signal-first architecture)
- **State management**: NgRx 19.1.0
- **UI**: TailwindCSS 4.1.10
- **i18n**: @ngx-translate/core 16.0.4
- **Cartographie / Mapping**: Leaflet, jsVectorMap
- **Backend (prévu / planned)**: NestJS
- **Données / Data**: Government APIs + CSV/JSON imports
- **Hébergement / Hosting**: 🇨🇦 100% Canadian servers and data storage

---

## 📍 Hébergement des données / Data hosting

All OpenG7 data is hosted exclusively on servers located in Canada to ensure:

- Sovereignty and compliance with Canadian laws  
- Protection against foreign jurisdictions  
- Enhanced security and confidentiality

## 📥 Installation & développement / Installation & development

### Prerequisites
- Node.js >= 20.x
- Yarn 3.x (nodeLinker: node-modules)
- Angular CLI 19.x

### Installation
```bash
# Clone the repository
git clone https://github.com/OpenG7/openg7-platform.git

# Install dependencies
yarn install

# Start development server
yarn start
```
