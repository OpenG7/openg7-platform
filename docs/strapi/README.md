# Documentation Strapi

La section `docs/strapi` centralise toutes les notes techniques, migrations et conventions liées au workspace officiel `strapi`.

- `workspaces.md` — aperçu du workspace `@openg7/strapi`, responsabilités et recommandations opérationnelles.
- `staging-data.md` — procédure pour alimenter l'environnement de staging sans réactiver les seeds automatiques.
- `database-scaling.md` — configuration base de données (SQLite ↔ Postgres), déploiement multi-instances et conteneurisation.

Ajoutez de nouveaux guides ici lorsque vous documentez une API, un processus de seed ou une configuration plugin.
